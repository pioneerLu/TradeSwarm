## 6. 数据使用说明

### 6.1 测试数据位置

所有测试数据存储在 `memory.db` 数据库的 `analyst_reports` 表中：

```sql
SELECT analyst_type, COUNT(*) as count 
FROM analyst_reports 
WHERE symbol='000001' 
GROUP BY analyst_type;
```

### 6.2 查询示例

**查询过去 7 天的历史报告**:
```sql
SELECT trade_date, report_content 
FROM analyst_reports
WHERE analyst_type='fundamentals' 
  AND symbol='000001'
  AND trade_date <= '2026-01-18'
  AND trade_date >= date('2026-01-18', '-7 days')
ORDER BY trade_date ASC;
```
